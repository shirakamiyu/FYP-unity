  # AR-Assisted Learning System
  
  以3D模型的方式呈現故事中的情節, 為孩子, **視覺與聽覺**的融合, 有助吸引孩子的興趣  
  與背景場景結合的音效, 讓孩子可以身臨其境了解所在地  
  以卡通風格設計的動物模型, 使孩子可以認識動物的特點
